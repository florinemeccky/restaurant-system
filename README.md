# Restaurant Reservation & Ordering System

A Laravel 12 restaurant management system built using:

- Laravel 12
- PHP
- MySQL
- Blade Templates
- Tailwind CSS
- Breeze Authentication

## Features

- Customer reservations
- Role-based authentication
- Admin dashboard
- Staff dashboard
- Menu browsing
- Reservation management

## Future Improvements

- Online payments
- Real-time table availability
- Mobile responsive UI
- Order tracking
- Qr codes
- Map technology
